# HTML Video Player

This is a basic HTML page with a video player.

## Usage

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Open the `index.html` file in your preferred web browser.

## Contents

- **index.html:** HTML file containing a video player with controls.
- **video.mp4:** Sample video file (you can replace it with your own).

## Features

- Displays a heading ("Hi").
- Includes a video player with controls.
- You can replace the `video.mp4` file with your own video.
